# Task 1 - Decision Tree Implementation

## Objective
Build and visualize a Decision Tree model using Scikit-learn to classify outcomes on a chosen dataset.

## Dataset Used
Iris Flower Dataset

## Technologies Used
- Python
- Scikit-learn
- Pandas
- Matplotlib

## Steps Performed
1. Imported required libraries
2. Loaded Iris dataset
3. Split dataset into training and testing data
4. Trained Decision Tree Classifier
5. Predicted outcomes
6. Calculated model accuracy
7. Visualized the Decision Tree

## Result
The Decision Tree model successfully classified flower species with high accuracy.
