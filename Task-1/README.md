## Project Overview

This project demonstrates the implementation of a Decision Tree Classifier using Scikit-learn on the Iris Flower Dataset.

The model is trained to classify flower species based on features such as sepal length, sepal width, petal length, and petal width.

---

## Technologies Used

- Python
- Scikit-learn
- Pandas
- Matplotlib
- Google Colab

---

## Dataset Used

- Iris Flower Dataset

The dataset contains three flower classes:
- Setosa
- Versicolor
- Virginica

---

## Steps Performed

1. Imported required libraries
2. Loaded the Iris dataset
3. Split dataset into training and testing data
4. Trained the Decision Tree model
5. Predicted test results
6. Calculated model accuracy
7. Visualized the Decision Tree

---

## Analysis

The Decision Tree model was successfully trained using the Iris dataset.

The model learned patterns from flower measurements and classified flower species with high accuracy.

The visualization shows how the model makes decisions step-by-step using conditions based on petal and sepal measurements.

From the analysis, it was observed that petal length and petal width are important features for classification.

The Decision Tree algorithm is easy to understand and visualize, making it suitable for beginner-level machine learning projects.

---

## Output

The model successfully classified flower species with high accuracy.

The Decision Tree visualization clearly shows how the model makes decisions based on feature conditions.
