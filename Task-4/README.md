## Project Overview

This project demonstrates a Movie Recommendation System using Cosine Similarity techniques for generating movie recommendations.

The system recommends similar movies by analyzing movie patterns and genre relationships using Machine Learning techniques.

---

## Technologies Used

- Python
- Pandas
- Scikit-learn
- CountVectorizer
- Cosine Similarity
- Google Colab

---

## Dataset Used

### MovieLens Movies Dataset (`movies.csv`)

The dataset contains:
- Movie Titles
- Movie Genres

### Dataset Source
https://files.grouplens.org/datasets/movielens/ml-latest-small.zip

---

## Steps Performed

- Imported required libraries
- Loaded movie dataset
- Displayed dataset information
- Handled missing values
- Converted movie genres into numerical vectors
- Calculated cosine similarity scores
- Built recommendation function
- Generated movie recommendations
- Tested recommendation system

---

## Analysis

The recommendation system was successfully implemented using Cosine Similarity techniques.

Movie genres were converted into numerical vectors using CountVectorizer.

Cosine Similarity was used to measure similarity scores between movies based on their genre relationships.

The recommendation model successfully generated similar movie suggestions for the selected input movie.

The project demonstrated how Machine Learning techniques can be used to build intelligent recommendation systems similar to platforms such as Netflix, Amazon, and YouTube.

The recommendation system successfully generated movie recommendations.
