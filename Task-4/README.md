## Project Overview

This project demonstrates a Movie Recommendation System using Content-Based Filtering and Cosine Similarity techniques.

The system recommends similar movies based on movie genres and content patterns.

---

## Technologies Used

- Python
- Pandas
- Scikit-learn
- CountVectorizer
- Cosine Similarity
- Google Colab

---

## Dataset Used

### MovieLens Movies Dataset (`movies.csv`)

The dataset contains:
- Movie Titles
- Movie Genres

### Dataset Source
https://files.grouplens.org/datasets/movielens/ml-latest-small.zip

---

## Steps Performed

- Imported required libraries
- Loaded movie dataset
- Displayed dataset information
- Handled missing values
- Converted movie genres into text vectors
- Calculated cosine similarity scores
- Built recommendation function
- Generated movie recommendations
- Tested recommendation system

---

## Analysis

The recommendation system was successfully implemented using Content-Based Filtering and Cosine Similarity.

Movie genres were converted into numerical vectors using CountVectorizer.

Cosine Similarity was used to calculate similarity scores between movies.

The system successfully recommended movies based on similar genres and content patterns.

The project demonstrated how recommendation systems are used in platforms such as Netflix, Amazon, and YouTube.

The recommendation system successfully generated similar movie recommendations.
