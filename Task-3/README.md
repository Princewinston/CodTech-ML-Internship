## Project Overview

This project demonstrates Image Classification using a Convolutional Neural Network (CNN) built with TensorFlow and Keras.

## Analysis

The CNN model was successfully implemented using TensorFlow and Keras on the Cats vs Dogs dataset.

The model learned image features such as edges, textures, and object patterns using convolutional layers.

MaxPooling layers reduced image dimensions and improved training efficiency.

The CNN model achieved approximately 70% accuracy on the testing dataset.

The project demonstrated the complete Deep Learning workflow including image preprocessing, CNN model training, image classification, and prediction.
